\begin{abstract}
Cluttered path clearing requires more than detecting blockages. It requires
selecting interventions that can enlarge a width-constrained corridor through
executable multi-robot interaction. Existing Navigation Among Movable Obstacles
methods mainly reason about obstacle rearrangement at an abstract geometric
level, whereas collaborative pushing methods emphasize local contact execution
without explicit guidance for multi-obstacle corridor construction. This paper
presents a physics-informed corridor hybrid search, abbreviated PICHS, for
collaborative path clearing. The search uses a W-clearance connectivity graph,
abbreviated WCCG, to determine whether a traversable corridor already exists
and, when it does not, to expose the reachable bottlenecks that limit progress.
For each selected bottleneck, only gap-conditioned pushing candidates are
generated for adjacent movable obstacles, and short-horizon physical
simulation selectively validates the induced interventions. In this
formulation, frontier gaps serve as an internal search interface rather than
as the primary algorithmic emphasis. Simulation and hardware results support
that PICHS improves path-clearing reliability while remaining consistent with
real multi-robot execution.
\end{abstract}
