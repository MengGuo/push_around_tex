\section{Conclusion} \label{sec:conclusion}
This work presents PushAround, a multi-robot path-clearing framework that
couples W--clearance reasoning, recursive push-task generation, contact-mode
optimization, and physics-informed hybrid search. The resulting planner jointly
selects blocking gaps, pushed obstacles, contact modes, and short-horizon
motions, while accepting only simulation-validated transitions. Simulations,
density-scaling tests, and hardware experiments demonstrate that the framework
constructs executable corridors in dense clutter. Future work will address state
estimation without external sensing, uncertain object parameters, and limited
robot visibility.
